About 3P

Progress Programmers Pal (or 3P)
is an OpenEdge ABL (formerly 4GL Progress) code editor running on notepad++
3P is an open-source (GPL V3 licence) project created and maintained by Julien Caillon [julien.caillon@gmail.com]
You can find out more about this project on its dedicated website http://jcaillon.github.io/3P/
Watch/Fork/Star the project on GitHub @ https://github.com/jcaillon/3P
